# Iris Flower Classifier - Multiple ML Models

A machine learning web app that compares multiple algorithms for iris species classification.

## Features
- Compare 4 ML algorithms: Random Forest, Decision Tree, SVM, KNN
- Real-time predictions with any model
- Model performance analysis with confusion matrices
- Interactive data exploration
